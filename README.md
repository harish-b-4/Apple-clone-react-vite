# Apple Website Clone (React + Tailwind CSS + Vite)

I created a modern Apple website UI clone using React, Tailwind CSS, and Vite. This project focuses on clean design, smooth layout, and responsive styling inspired by Apple’s official website.

# Features

Fully responsive first page UI

Built with React + Vite for fast performance

Styled using Tailwind CSS for modern and clean design

Smooth scrolling and layout similar to Apple’s homepage

Well-structured and easy-to-understand code

# Tech Stack

React

Vite

Tailwind CSS

# Purpose

This project was built to practice frontend development, focusing on UI design, layout techniques, and responsiveness using Tailwind CSS.
